**Автоматический парсер новостей из RSS-фидов российских СМИ**

VertexNewsParser — это Go-приложение, которое периодически собирает новости из ведущих российских новостных источников (ТАСС, РИА, Лента.ру и др.), очищает их и сохраняет в базу данных PostgreSQL, избегая дубликатов.

Возможности
- Парсинг RSS-фидов из 11+ источников
- Параллельная обработка источников (goroutines)
- Автоматическая дедупликация новостей по заголовку
- Очистка HTML-разметки в описаниях
- Полностью контейнеризировано с помощью Docker
- Гибкая конфигурация через `.env`
